# wubigendaqi
模仿添雨跟打器，感谢。

qt写的五笔跟打器，支持在windows, mac, linux上运行。

没有静态编译，暂时需要有安装qt的环境才能运行。

- [x] 顺序，随机发文
- [x] 自动切换文段
